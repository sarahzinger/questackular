Chrome Extension + Web Application

Questackular is a web application and chrome extension which allows users to create and join virtual treasure hunts. Join a quest, download the chrome extension, and begin your learning journey.

Each step in a quest, will launch you to a new page, article, or website designed to enhance your learning. If you're able to answer the riddle or question associated with the page you will earn points which can be redeemed for virtual prizes.

Educators can create their own quests for their students and get instant feedback on how their students are progressing on their own educational journeys.
